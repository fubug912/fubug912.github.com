前言

其实这个博客已经搭建很久了,只是那段时间比较忙没有时间来管理这个博客也就什么都没有写.最近这段时间有些空闲了,我才想起这个博客,发现啥也没有,就想写一下东西了,不过本人现在就是一个小白啥都不会.想了好几天,才想出要写这篇文章,这篇文章也是本人第一次写的文章,有不对的地方望大神们指点!

我与hexo

我是在大概一年前的时候开始接触hexo的,当时,是因为脑袋不知道发啥神经想建个网站所以的开始了,先是开始接触Wordpress,后面发现配置太麻烦,开网站的时候加载又慢(服务器耗不起- -.),因为我是一个小白不会写主题,好看的主题又需要购买,所以就只好放弃了(其实是因为小白我是一个屌丝,穷买不起服务器和主题.土豪可以玩).后来就遇到我hexo.才发现有这么好的东西.下面来介绍下hexo的一些特点(个人认为,土豪,大神们勿喷)

加载速度快

因为hexo是基于Node.js.所以它加载比Wordpress快了不少.

支持Markdown

是的.hexo支持Markdown,什么您不知道Markdown是什么?好吧,我告诉您这篇文章就是用这个写的.至于它的优点请去它的Wiki上看吧.哦,国内可能访问不了,在这我给一个百度百科上的网址不过还是有条件的朋友上Wiki看比较好.至于什么上我就不说了,懂的自然会懂,不懂就….

简单

简单,是因为它不需要购买服务器.甚至不需要购买域名(github上会提供免费的,当然会很长如果您不建议的话可以用.)不过还是推荐您购买个域名(毕竟现在域名也不贵多少了,万一那个域名变值钱了呢.哈哈哈),需要安装的东西也不多,不像Wordpress,不是搭建这个就是配置那个..麻烦.

强大的插件

hexo有着强大的插件系统,可以让其拥有强大的扩展性.

其实说这么多,总结起来就是:快速,简单,可扩展!

准备工作

由于我现在用的是Mac系统所以我主要讲下Mac系统上的安装同时我也会讲一下Windows系统上的安装毕竟用的人比较多(其实我第一次安装hexo就是在win上安装’滑稽’)

Mac

在Mac系统上,需要以下工具:

  1. homebrew (可选)
  2. node.js
  3. npm (node.js已集成)

  4. hexo

    注意:node.js已经集成npm了,不需要重新安装npm了.

node.js

在Mac下有几种方法,下面主要介绍两种.

  1. 直接到官网下载安装
  2. 使用brew命令安装node.js

因为使用brew安装,有可能失败(因为有墙).所以我推荐使用第一种方法.喜欢挑战的人也别担心,上面的两种方法我都会讲.

在官网下载node.js并安装

首先,先进入官网(废话),看图

这两个可以根据情况下载,下载完成后,打开安装就可以了,安装完成后会显示一些安装路径在上面我们就可以发现,它顺便帮你安装npm了.图我就不发了(懒),因为我已经安装过了.

检测安装是否成功可以用下面的方法:

打开你的终端(在应用列表里的其他目录上,我的在这里面不知道您们的是否一样找不到的话可以搜索),输入node -v并回车,你会看到如下:

➜  ~ node -v
v6.3.0

说明安装成功了,如果不是请重新试试或者试试下面的方法.

homebrew(brew) 安装 node.js

homebrew 是一个Mac osx 系统下的安装或卸载软件的软件.详细介绍请到官网查看.

首先,先安装homebrew(又是废话).

安装homebrew 很简单,只需要在终端上输入并回车下面的命令,等待完成就好了.

/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

ps: 上面的命令是我写博客的时候(2016.8)放上去的不一定是最新的,一切已官网的命令为准,如果无法安装请到官网上去获取命令,在官网的获取Homebrew下面.

安装完成后可以用brew -v命令检测安装是否成功,结果如下.

➜  ~ brew -v
Homebrew 0.9.9 (git revision 0be20; last commit 2016-07-26)
Homebrew/homebrew-core (git revision 4765b; last commit 2016-07-26)

下面给出homebrew一些常用的命令

brew search wget    #搜索wget
brew info wget      #显示wget的详细信息
brew install wget   #安装wget
brew unintsall wget #卸载wget
brew list wget      #列出已安装的软件

好了,下面开始用brew命令安装node.js.
在终端输入brew install node.js并回车,等待命令执行完成,用node -v命令查看是否安装成功.

安装hexo

可以进入官网点击文档查看详细的安装方法.

直接在终端上执行npm install -g hexo-cli 即可完成安装.如果失败,请尝试安装 Xcode,在App Store中搜索Xcode点击安装.安装完成后启动并进入Preferences -> Download -> Command Line Tools -> Install 安装命令行工具,然后在命令行再执行npm install -g hexo-cli.

搭建博客

搭建本地博客很简单只需要依次在终端输入以下命令即可.

hexo init ~/blog #初始化. 这个目录(~/blog)可以自行修改,这个是您博客的目录
cd ~/blog        #进入博客目录
npm install      #安装

安装完后在安装目录如下

.
├── _config.yml
├── package.json
├── scaffolds
├── source
|   ├── _drafts
|   └── _posts
└── themes

详细安装教程和目录详解请看官网进入文档 > 建站查看

Linux

Linux系统上我就不详细讲了,使用Linux的一般都已经是大神了,可以查看官网文档 > 开始使用里的教程搭建就OK了,下面引用了官网的内容

####安装 Git

Linux (Ubuntu, Debian)：sudo apt-get install git-core

Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core

####安装 Node.js

安装 Node.js 的最佳方式是使用 nvm。

cURL:

$ curl https://raw.github.com/creationix/nvm/master/install.sh | sh

Wget:

$ wget -qO- https://raw.github.com/creationix/nvm/master/install.sh | sh

安装完成后,重启终端并执行下列命令即可安装 Node.js.

$ nvm install stable

或者您也可以下载安装程序来安装。

安装hexo

所有必备的应用程序安装完成后，即可使用 npm 安装 Hexo。

$ npm install -g hexo-cli

按照上面的就可以安装成功了.
